# French Cuisine - книга рецептов французской кухни

Данное приложение является курсовой работой.

Его основные задачи: 
  1. Просмотр всей базы данных рецептур блюд
  2. Поиск по категории блюда, по сложности приготовления и по
	   ингредиентам
  3. Добавление новых рецептов с прикреплением новой картинки
  4. Возможность редактировать созданные элементы
  5. Блюдо дня, которое будет выводиться на главный экран
